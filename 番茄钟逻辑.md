### 一、页面布局描述（从上到下）

**1. 顶部：任务记录区**

- **位置**：页面最上方，占约 20% 的高度。
- **内容**：一个单行文本输入框，带有 placeholder 提示“当前要做的事情...”。
- **功能**：用户可以随时输入或修改当前专注的任务名称。这个任务名称会在计时器下方显示，作为当前的专注目标。

**2. 中部：核心计时器区**

- **位置**：页面正中央，视觉焦点。
- **内容**：
    - 一个大的圆形进度环（SVG 圆环）。
    - 圆环内部居中显示倒计时数字（如 `25:00`），字体超大且等宽。
    - 圆环正下方显示当前状态标签（如 ` 专注中` 或 ` 短休息`）。
    - 再下方显示当前正在执行的任务名称（从顶部输入框同步过来）。
- **视觉反馈**：
    - 专注状态：圆环颜色为番茄红/橙色，背景微红。
    - 休息状态：圆环颜色为薄荷绿/海蓝色，背景微绿。

**3. 底部：设置与控制区**

- **位置**：页面最下方，占约 20% 的高度。
- **内容**：
    - **连开设置**：一个下拉选择框或数字步进器，标签为“连续专注轮数”，可选值为 1~8（默认 4）。这决定了完成多少个番茄钟后触发长休息。
    - **控制按钮组**：水平排列三个按钮：

### 二、操作逻辑与流程

#### 1. 初始化状态

- 页面加载后，计时器显示 `25:00`，状态为“准备专注”，任务为空（用户需先输入）。
- 连开设置默认为 4 轮。
- 当前已完成轮数计数器 = 0。

#### 2. 开始专注

- 用户点击「开始」按钮。
- 计时器开始倒计时，圆环动画启动。
- 状态变为“ 专注中”，网页标题同步显示 `[25:00] 专注中`。
- 如果用户未输入任务，弹出轻提示：“请先输入当前任务”。

#### 3. 专注结束（25分钟到）

- 倒计时归零，自动播放语音提示音（如清脆的铃声 + 语音播报：“专注时间结束，休息一下吧”）。
- 状态自动切换为“ 短休息”，计时器自动设为 `05:00` 并开始倒计时。
- 圆环颜色变为绿色，网页标题变为 `[05:00] 短休息`。
- 已完成轮数计数器 +1。

#### 4. 短休息结束

- 倒计时归零，自动播放语音提示音（如柔和的提示音 + 语音播报：“休息结束，继续加油！”）。
- **判断是否需要长休息**：
    - 如果已完成轮数 < 连开设置值 → 自动进入下一轮专注（25:00），重复步骤 2。
    - 如果已完成轮数 = 连开设置值 → 自动进入长休息（15:00），状态变为“ 长休息”，圆环颜色变为深蓝色。

#### 5. 长休息结束

- 倒计时归零，播放语音提示音：“长休息结束，新一轮番茄钟即将开始！”
- 已完成轮数计数器归零。
- 自动进入下一轮专注（25:00），重复步骤 2。

#### 6. 暂停与继续

- 专注或休息期间点击「暂停」→ 计时器停止，按钮变为“继续”。
- 再次点击「继续」→ 从暂停处恢复倒计时。

#### 7. 重置

- 点击「重置」→ 计时器回到当前阶段的初始时间（专注回到 25:00，休息回到对应休息时间），已完成轮数不变。

#### 8. 跳过

- 点击「跳过」→ 立即结束当前阶段，按正常流转逻辑进入下一阶段（专注→短休息，短休息→专注/长休息）。

### 三、关键数据状态

| 变量 | 说明 |
| ------ |------ |
| `currentTask` | 当前任务名称（字符串） |
| `phase` | 当前阶段：focus / shortBreak / longBreak |
| `remainingSeconds` | 剩余秒数 |
| `completedRounds` | 已完成的专注轮数 |
| `targetRounds` | 连开设置的目标轮数 |
| `isRunning` | 是否正在计时 |
